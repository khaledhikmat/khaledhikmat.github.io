This site is about the Symphony for .NET backend product! It documents all aspects of the products and provides links for further reading. The main objective is to provide a one-stop shop for all information about the Symphony backend system.

The site is broken down into the following main sections:

  • Blogs - lists all the blogs that are relavant to this project. There is some technical information and know-how.
  • Resources - lists all the files, presentaions, source code and binaries that are related to this project.
  • User Stories - lists in groups all the user stories related to this project.
